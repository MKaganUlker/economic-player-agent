# Economic Player Agent

> An autonomous AI agent that continuously improves financial decisions through reasoning, tool use, simulation, and verification.

## Overview

Most AI agents make a single decision and execute it.

**Economic Player Agent** treats decision making as an iterative process:

1. Observe the current economic environment.
2. Reason about possible actions.
3. Execute actions through tools.
4. Observe the outcome.
5. Repeat.

The goal is simple:

> **Maximize long-term net worth.**

Unlike rule-based simulations, every decision is generated by a local LLM, while the environment remains fully deterministic and reproducible.

---

## Current Features

* Local LLM powered by Ollama
* ReAct-style reasoning
* Tool-based action execution
* Economic environment simulation
* Structured JSON decision outputs
* Extensible architecture

Current tools:

* Invest
* Save money
* Learn skills
* Do nothing

---

## Why This Project?

Most AI demos stop after generating an answer.

This project focuses on autonomous decision making by separating:

* reasoning
* environment
* tool execution
* simulation

This makes the system deterministic, testable, and suitable for future multi-agent experiments.
